# Quick Start Guide - Nyero AVR System

## 🚀 Setup & Installation

### Prerequisites

- **Node.js** (LTS version) - Download from https://nodejs.org
- A modern web browser (Chrome, Firefox, Edge, Safari)

### Windows Setup

1. Extract the project folder
2. **First time only**: Run `setup.bat` to install dependencies
3. Then run `start_local.bat` to start the server
4. The app will open automatically at `http://localhost:8000/app.html`

### Linux/Mac Setup

1. Extract the project folder
2. Open terminal in the project folder
3. **First time only**: Run `npm install` to install dependencies
4. Then run `bash start_local.sh` to start the server
5. The app will open automatically at `http://localhost:8000/app.html`

## 🎮 Getting Started in 5 Minutes

### Step 1: Create Your Account (30 seconds)

1. Open the app
2. Enter a username (3-20 characters)
3. Create a strong password (8+ characters)
4. Click "Create account"

**Or**: Click "Continue as Guest" to explore without an account

---

### Step 2: Learn the Controls (1 minute)

**Desktop**:

- `WASD` or arrow keys = Move
- Mouse drag = Look around
- Scroll = Move forward/backward
- Number keys `1-0, Q, W` = Quick teleport to paintings

**Mobile**:

- Drag screen = Look around
- On-screen buttons = Move forward/back/left/right

---

### Step 3: Explore Your First Painting (2 minutes)

1. Look at the navigation panel (top-left)
2. Click any painting button (e.g., "Nyero 1")
3. You'll teleport there automatically
4. Click on the painting to learn about it
5. A detailed view appears with description

---

### Step 4: Enable Guided Tour (1 minute)

For a structured experience:

1. Click "⚙️ Settings" button
2. Find "Guided Tour" toggle
3. Click to enable
4. Follow the on-screen guidance
5. You'll visit all 11 paintings in sequence

---

### Step 5: Track Your Progress (30 seconds)

Your exploration progress appears in the **top-right corner**:

- Shows paintings visited: `X/11`
- Visual progress bar
- Updates automatically as you visit each site

### Use the Minimap

1. Open Settings
2. Enable "Show Minimap"
3. Watch the bottom-right corner
4. Blue square = you
5. Colored dots = paintings
6. White arrow = facing direction

### Search Feature

1. Click search bar at the top
2. Type painting name or keyword
3. Click result to go there
4. Examples: "sun", "circles", "ritual"

### Accessibility

- **Text too small?** Settings → Text Size
- **Need labels?** Settings → Hotspot Labels toggle
- **Want beacon?** Settings → Pointer Beacon toggle
- **Prefer less motion?** Disable Particle Effects

### Performance

If you experience lag:

1. Open Settings
2. Disable "Particle Effects"
3. Disable "Shadows"
4. Check "Performance Monitor" to see FPS

## ❓ Common Questions

### Q: How do I save my progress?

**A**: Your progress saves automatically when you visit paintings. Login with the same account to see your progress.

**A**: Yes! Full mobile support with touch controls and responsive design.

### Q: What if I get stuck?

**A**: Press `H` or click "❓ Help" for comprehensive help documentation.

### Q: Can I make the text larger?

**A**: Yes! Settings → Accessibility → Text Size (adjust slider).

### Q: How do I exit fullscreen?

**A**: Press `ESC` or click Settings → toggle Fullscreen again.

### Q: Will my data be saved?

**A**: Yes! All settings, progress, and preferences save locally on your device.

### Q: Can I share my progress?

**A**: Your progress is private. Log in from another device with your account to see your stats.

### Q: What's the fastest way to visit all paintings?

**A**: Use keyboard shortcuts (1-0, Q, W) to teleport directly. Or enable Guided Tour for automatic navigation.

## 🔧 Troubleshooting

### Issue: Controls not working

- **Fix**: Click in the scene first to focus it
- Then try WASD or arrow keys

### Issue: Paintings look blurry

- **Fix**: Adjust graphics settings
- Settings → Graphics → Toggle Shadows/Fog

### Issue: Too slow/laggy

- **Fix**: Disable particle effects
- Settings → Graphics → Toggle Particle Effects OFF

### Issue: Can't find a painting

- **Fix**: Use search feature
- Type painting name in search bar at top

### Issue: Lost progress after closing

- **Fix**: Make sure you're logged in
- New accounts auto-save on each visit
