# Nyero AVR System - Enhanced Features

## 🎯 New Interactive Features Added

### 1. **Settings & Preferences Panel** ⚙️

- **Graphics Options**: Toggle particle effects, shadows, and fog
- **Audio Controls**: Volume slider and ambient sound toggle
- **Accessibility Settings**:
  - Hotspot labels visibility
  - Pointer beacon toggle
  - Text size adjustment (0.8x - 1.5x)
- **Gameplay Options**:
  - Guided tour mode
  - Minimap display
  - Performance monitor
  - Fullscreen toggle

**How to access**: Click the "⚙️ Settings" button or press `O`

---

### 2. **Guided Tour Mode** 🗺️

- Automatically guides you through all 11 rock painting sites
- Step-by-step navigation with descriptions
- Educational experience for new visitors
- Toast notifications for each location

**How to use**:

1. Open Settings
2. Toggle "Guided Tour" on
3. The system will automatically navigate you through each painting
4. Progress shown in real-time

---

### 3. **Progress Tracking** 📊

- Real-time display of explored paintings (top-right corner)
- Shows: X/11 paintings visited
- Visual progress bar
- Automatically updated when you click on paintings

**Features**:

- Persistent progress tracking (saved locally)
- Motivational notifications
- Complete all 11 sites for full exploration!

---

### 4. **Search & Filter** 🔍

- Quick search bar at the top of the screen
- Search by painting name or keywords
- Instant results with direct navigation
- Works with tags like: "celestial", "ritual", "geometric", etc.

**Example searches**:

- "Sun" → finds Sun Motif
- "circles" → finds Concentric Circles
- "ritual" → finds Ritual Motifs

---

### 5. **Interactive Minimap** 🗺️

- Live 2D map of your location in the Nyero site
- Shows all 11 painting locations as colored dots
- Your position and viewing direction indicator
- Toggle on/off from Settings

**What you see**:

- Blue square = Your player position
- White arrow = Your viewing direction
- Colored dots = Rock painting locations
- Grid borders = Scene boundaries

---

### 6. **Favorites System** ⭐

- Mark paintings as your favorites
- Save for later reference

- Access from navigation bar

**How to use**:

- Paintings marked as visited are tracked

- Share favorites with others

## 🆘 Troubleshooting

**Q: Progress not saving?**

- Clear browser cache and reload
- Check if localStorage is enabled

**Q: Minimap not showing?**

- Enable it in Settings → Show Minimap
- Requires JavaScript enabled

**Q: Performance slow?**

- Open Performance Monitor to check FPS
- Disable particles and shadows in Settings
- Close other browser tabs

**Q: Text too small/large?**

- Use Settings → Accessibility → Text Size
- Adjust slider to comfortable level
